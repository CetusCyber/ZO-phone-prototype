# 📱 ZO Orchestrator – Mobiler Prototyp

## 🧾 Kurzbeschreibung
* 📱 **ZO** ist ein experimenteller Prototyp im Rahmen eines größeren, in Entwicklung befindlichen Projekts.
* 🧪 Ziel ist es, das Konzept praktisch auf echter Hardware mit begrenzten Ressourcen zu testen.
* 🚀 Diese mobile Version ist der erste Schritt zur Einführung des vollständigen ZO-Projekts aus Syrien.

---

## 🎯 Projektziele
* 🔧 Verwandlung eines Smartphones in eine technische Experimentierplattform.
* 🤖 Testen von KI- und Cybersecurity-Funktionen auf mobilen Endgeräten.
* 🐧 Integration verschiedener Ebenen:
    * Linux-Umgebungen
    * Sicherheitstools
    * Lokale KI-Modelle

---

## ⚙️ Hardware
* **Gerät:** Nothing Phone (2a)
* **Arbeitsspeicher:** 8GB RAM

---

## 🧱 System-Stack
1. **Basis:** Android mit **Termux** als Terminal-Emulator.
2. **Linux-Ebene:** **Ubuntu** (via `proot-distro`) für eine vollständige Linux-Umgebung.
3. **Lokale KI:** **Ollama** innerhalb von Ubuntu für die Ausführung lokaler Sprachmodelle.
4. **Sicherheitsebene:** **NetHunter**-Integration für Sicherheitsanalysen und Bildungszwecke.
